# Book Scraper

A Python web scraper that automatically collects book titles and prices from books.toscrape.com and saves them to Excel.

## What it does
- Visits the website automatically
- Scrapes all book titles and prices
- Saves the data to an Excel file

## Tools used
- Python
- requests
- BeautifulSoup4
- openpyxl

## How to run
1. Install requirements: `pip install requests beautifulsoup4 openpyxl`
2. Run: `python scraper.py`
